Атрибут accesskey - это тег который наводить фокус на какой-то тег, сочетанием клавиш.Работает с комбинацией клавиш ALT+S ( но можно настроить ALT+любая кнопка от a-z и 1-9), и к тому же можно эту ссылку сделать невидимой, тоесть оставить тег "а" пустым, а комбинацию клавиш на ссылку оставить, допустим щас ты ссылки не видишь, но она есть на сайте, и если ты нажмешь ALT+S то перейдёшь по ссылке.

Атрибут contenteditable позволяет пользователю редактировать блок, допустим здесь пользователь может сам редактировать этот блок, если по блоку щелкнет мышкой.

id - это уникальное имя элемента, которое используется для изменения его стиля, и обращение к нему через скрипты, также по id в базу данных идут какие-то данные. Айдишник всегад должен быть унивкальным, он не должен повторяться. Также в его написании не должны использоваться пробелы, цифры, символы подчёркивания, дефис.

lang - этот атрибут указывает ан язык в теги. Так как кавычки и другие спец. символы выглядят по разному на разном языке+гугл переводчик при автоматическом переводе будет переводить на тот язык, который указан в теге этом. К примеру как выглядят кавычки ан разных языках.

Spellcheck - это атрибут проверки правописания. Действует только в тегах input и textarea а также в элементах с атрибутом contenteditable, ещё стоит упомянуть что проверку правописания можно отключить.

В теге Style - мы прописываем css свойства которые хотим придать html - тегу. Но так делают только дауны, лучше делать отдельным файлом стили цсс и через тег хед подключать его к html документу.

Тег tabindex - устанавливает порядок получения фокуса между элементами с помощью клавиши tab, а также может убрать фокус на теге при табе.
